Image Convertion

This program can convert an image to an XPM image. The resulting XPM image will have up to 64 different colors. The algorithm chooses the color that best fits the current pixel being evaluated.

NOTE: This program considers that the color with index 5 is the transparency color.


The format of the supported images are: JPEG and PNG.

How to use:
  1. Press the 'Add Image' button to add images to be converted. Multiple images can be added.
  2. Check 'generate code-format' if you want the output to be in a format recognized in code (for copy-paste purposes).
  3. Press 'Output Path" to specifiy the folder where the generated files be placed.
  4. Click the 'Convert Images' button to convert all images in the list.

The convertion process uses multiple threads by default (the number of threads will be equal to half of the logical cores available). You can change the amount of threads used and whether to use multiple threads in the settings section. It is not recommended that you disable multiple thread usage, as it will slow the conversion process greatly.



Image Edition (Rotate, Scaling & Simple Edit)

You can rotate and scale an image by going on the 'Transform' option on the menu bar and selecting the desired transformation.

This software also allows simple edition of an XPM image. You can do so by pressing the 'Edit XPM' button. After choosing the desired XPM to edit, a window will open containing the image to edit and the 64 colors you can use to paint it.



Image Preview

You can view an XPM image by pressing the 'Preview XPM' button. After selecting the desired XPM file to preview, a window will open containing the XPM image.
